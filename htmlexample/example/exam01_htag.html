나의 첫 번째 웹사이트

    
소개

    
이 웹사이트는 HTML 태그의 사용법을 연습하기 위한 것입니다.

    
목표

    
HTML은 웹 문서의 구조를 만드는 마크업 언어입니다.

    
웹 개발 기본 요소

    
1.HTML

    
HTML은 웹 문서의 구조를 만드는 마크업 언어입니다.

    
2.CSS

    
CSS는 웹 페이지의 스타일과 레이아웃을 정의하는 데 사용됩니다.

    
3.JavaScript

    
JavaScript는 웹 페이지에 동적인 기능을 추가하는 데 사용됩니다.

    웹문서 구현(클라이언트, front-end)
    
     html: 문서의 구조를 작성
    
     css: 문서에 스타일 적용
    
     JavaScript: 문서를 동적으로 처리한다.
